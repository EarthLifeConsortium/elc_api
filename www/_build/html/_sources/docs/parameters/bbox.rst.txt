``bbox=POLYGON((...))``
    Geographic bounding box for constraining the response. Must be a strictly formated OGC WKT Polygon. The unaffiliated tool `OpenStreetMap WKT Playground <https://clydedacruz.github.io/openstreetmap-wkt-playground>`_ may be useful in graphically creating a polygon. Paste the generated WKT string into the ``bbox`` field of the ELC API. Type: `str`
